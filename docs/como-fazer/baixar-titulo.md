# Como baixar um título

## Pré-requisitos

<div class="wc-prereq-list" markdown>

- Título financeiro localizado
- Conta, forma de pagamento e data de baixa definidos
- Valor pago ou recebido conferido

</div>

## Permissões

--8<-- "shared/avisos/permissoes.md"

## Caminho
`Financeiro > Contas a Receber / Contas a Pagar`.

## Como fazer

1. Acesse a rotina de contas a receber ou contas a pagar.
2. Localize o título.
3. Abra o registro.
4. Acione a opção de baixa.
5. Informe conta, data, valor e forma de pagamento.
6. Revise juros, descontos ou acréscimos.
7. Confirme a baixa.

## Quando utilizar

Use quando um cliente pagou, um fornecedor foi pago ou quando a baixa precisa ser registrada manualmente.
