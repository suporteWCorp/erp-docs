# Como cadastrar um fornecedor

## Pré-requisitos

- CNPJ, CPF ou dados cadastrais do fornecedor.
- Confirmação de que o fornecedor ainda não está cadastrado.
- Acesso à tela [Manual > Fornecedores > Fornecedores](../fornecedores/fornecedores.md){: target="_blank" rel="noopener" }.

--8<-- "shared/avisos/permissoes.md"

## Onde encontrar

Caminho: `Fornecedores > Fornecedores`.

## Como fazer

1. Acesse **Fornecedores > Fornecedores**.
2. Inicie um novo cadastro.
3. Informe CNPJ, CPF ou dados principais.
4. Complete dados fiscais, contato e endereço.
5. Revise os campos obrigatórios.
6. Salve o cadastro.

## Erros comuns

Caso encontre algum dos erros abaixo, clique para visualizar a causa e como resolver.

- Fornecedor duplicado — **Pendente**
- Dados fiscais incompletos — **Pendente**

## Quando utilizar

Use quando uma empresa ou pessoa ainda não existir como fornecedor no WCorp.
