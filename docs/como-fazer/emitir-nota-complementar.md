# Como emitir uma nota complementar

## Pré-requisitos

<div class="wc-prereq-list" markdown>

- NF-e original localizada
- Dados do complemento definidos
- Natureza de operação adequada para complemento cadastrada<br>
  [Como cadastrar uma natureza de operação.](cadastrar-natureza-operacao.md){: target="_blank" rel="noopener" }

</div>

## Avisos

--8<-- "shared/avisos/validacao-fiscal.md"

## Permissões

--8<-- "shared/avisos/permissoes.md"

## Caminho
`Faturamento > Nota Fiscal`.

## Como fazer

1. Acesse **Faturamento > Nota Fiscal**.
2. Crie uma nova nota complementar.
3. Informe o participante da operação.
4. Referencie a NF-e original.
5. Informe valores, impostos ou dados complementares.
6. Confira a natureza de operação.
7. Salve e transmita a nota.
8. Confira o retorno da SEFAZ.

## Quando utilizar

Use quando a operação original precisa de complemento fiscal autorizado pela regra tributária aplicável.
