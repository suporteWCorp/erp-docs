# Como cadastrar cliente

## Antes de começar

- Tenha o CNPJ ou dados cadastrais do cliente.
- Confira se o cliente ainda não existe no cadastro.

## Caminho no ERP

`Comercial > Clientes`

## Passo a passo rápido

1. Acesse a aba **Comercial**.
2. Clique em **Clientes**.
3. Inicie um novo cadastro.
4. Informe o **CNPJ** e clique na **lupa** para buscar os dados automaticamente.
5. Confira os dados retornados.
6. Complete os campos que faltarem.
7. Clique em **Salvar**.

!!! info "Importante"
    Caso seja um CNPJ recém-cadastrado, a atualização na base da SEFAZ pode levar mais de 30 dias. Nesses casos, pode ser necessário preencher as informações manualmente.
