## Como fazer

1. Acesse **Comercial > Orçamento**.
2. Crie um novo orçamento.
3. Informe o cliente.
4. Adicione materiais, quantidades e valores.
5. Informe condição de pagamento, prazo e observações.
6. Salve o orçamento.

## Quando utilizar

Use quando o cliente ainda está avaliando preços, quantidades, prazos ou condições comerciais.
