# Glossário

| Termo | Significado |
| --- | --- |
| ERP | Sistema integrado de gestão empresarial |
| Módulo | Área funcional do sistema |
| Rotina | Funcionalidade ou processo executado no WCorp |
| Título | Registro financeiro a pagar ou receber |
| Baixa | Confirmação de pagamento ou recebimento |
| Natureza de operação | Classificação fiscal ou operacional de uma movimentação |
| NCM | Classificação fiscal de mercadorias |
| SEFAZ | Secretaria da Fazenda responsável pela autorização fiscal |
