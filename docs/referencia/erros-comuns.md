# Erros comuns

Use esta página como base de conhecimento para mensagens recorrentes do Sistema.

| Mensagem | Onde ocorre | Causa provável | Como resolver |
| --- | --- | --- | --- |
| Cliente bloqueado | Pedido de venda | Restrição no cadastro | Validar motivo do bloqueio e solicitar liberação |
| Produto sem saldo | Pedido ou faturamento | Quantidade indisponível | Conferir estoque e almoxarifado |
| Rejeição fiscal | Nota fiscal | Dados fiscais inconsistentes | Ler retorno da SEFAZ e corrigir origem do dado |

## Como registrar um novo erro

1. Copie a mensagem exatamente como aparece.
2. Informe em qual tela ou processo ocorreu.
3. Descreva o que o usuário estava tentando fazer.
4. Registre a causa confirmada.
5. Inclua o procedimento de correção.
