# Processo

## Objetivo

Cadastrar ou consultar processos produtivos.

## Print da tela com caminho

$Tab > Processo

## Quando usar

Use esta rotina quando houver necessidade de executar ou consultar processos relacionados a **Processo**.

## Passo a passo

1. Acesse a aba **Produção**.
2. Clique em **Processo**.
3. Informe os filtros ou dados necessários.
4. Confira as informações antes de salvar, confirmar ou finalizar.
5. Finalize conforme a ação disponível na tela.

### Campos principais

| Campo | Descrição | Observações |
| --- | --- | --- |
| A definir | A definir | A definir |

## Dúvidas frequentes

| Dúvida | Orientação |
| --- | --- |
| Quando devo usar esta rotina? | Use quando o processo estiver relacionado a **Processo**. |
| O que devo conferir antes de salvar? | Confira os dados obrigatórios, filtros e mensagens exibidas pelo sistema. |
