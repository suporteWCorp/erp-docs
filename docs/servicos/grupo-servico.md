# Grupo de Serviço

## Objetivo

Cadastrar ou consultar grupos usados para classificar serviços.

## Print da tela com caminho

$Tab > Grupo de Serviço

## Quando usar

Use esta rotina quando houver necessidade de executar ou consultar processos relacionados a **Grupo de Serviço**.

## Passo a passo

1. Acesse a aba **Serviços**.
2. Clique em **Grupo de Serviço**.
3. Informe os filtros ou dados necessários.
4. Confira as informações antes de salvar, confirmar ou finalizar.
5. Finalize conforme a ação disponível na tela.

### Campos principais

| Campo | Descrição | Observações |
| --- | --- | --- |
| A definir | A definir | A definir |

| Dúvida | Orientação |
