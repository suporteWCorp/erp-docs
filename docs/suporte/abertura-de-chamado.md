# Abertura de chamado

Use este modelo para abrir chamados internos ou encaminhar casos para análise técnica.

## Modelo de chamado

```text
Título: [Módulo] Resumo curto do problema

Empresa/cliente:
Usuário afetado:
Módulo:
Tela/caminho:
Registro afetado:
Data e horário aproximados:

O que o usuário tentou fazer:

O que aconteceu:

O que era esperado:

Mensagem exibida:

Passos para reproduzir:
1.
2.
3.

Evidências anexadas:
- Print da tela
- Print da mensagem
- Número do pedido/nota/título/material
- Logs ou retorno técnico, se houver
```

## Quando encaminhar para desenvolvimento

Encaminhe quando o problema for reproduzível, não houver configuração evidente que explique o comportamento e a rotina estiver diferente do resultado esperado.

Antes de encaminhar, confirme:

- [ ] O comportamento foi reproduzido.
- [ ] O caso possui dados suficientes para análise.
- [ ] O impacto foi informado.
- [ ] A urgência foi justificada.
- [ ] Já foi verificado se existe erro conhecido na base.
