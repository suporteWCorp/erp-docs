# Layout de Ficha Técnica

## Objetivo

Configurar ou consultar layouts de ficha técnica.

## Print da tela com caminho

$Tab > Layout de Ficha Técnica

## Quando usar

Use esta rotina quando houver necessidade de executar ou consultar processos relacionados a **Layout de Ficha Técnica**.

## Passo a passo

1. Acesse a aba **Administração**.
2. Clique em **Layout de Ficha Técnica**.
3. Informe os filtros ou dados necessários.
4. Confira as informações antes de salvar, confirmar ou finalizar.
5. Finalize conforme a ação disponível na tela.

### Campos principais

| Campo | Descrição | Observações |
| --- | --- | --- |
| A definir | A definir | A definir |
